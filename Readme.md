# SheCode Basics 


## HTML
Headings
```html
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
```
Paragraphs
```html
<p>Hello World!</p>

```
Lists
```html
<ul>
    <li>Hello!</li>
    <li>World!</li>
</ul>

<ol>
    <li>Hello!</li>
    <li>World!</li>
</ol>
```

Styling
```html
<em>Hello world italic!</em>
<strong>Hello world bold!</strong>
```
Self closing tags
```html
<hr />
<br />
```

ShortCuts
- ctrl + space bar -> to write all the html doc template
